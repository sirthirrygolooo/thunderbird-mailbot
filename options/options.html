Mailbot CEMagik

Extension Thunderbird — Google Gemini AI

Configuration de l'API Gemini

Pour utiliser Mailbot CEMagik, vous devez configurer votre clé API Google Gemini.

Votre clé API est stockée localement et n'est jamais partagée.

Comment obtenir une clé API Gemini ?

  1. Rendez-vous sur Google AI Studio
  2. Connectez-vous avec votre compte Google
  3. Cliquez sur « Get API Key » ou « Créer une clé API »
  4. Copiez la clé générée et collez-la ci-dessus

Configuration de l'API Mistral

Pour utiliser Mailbot CEMagik avec Mistral, vous devez configurer votre clé API Mistral.

Votre clé API est stockée localement et n'est jamais partagée.

Comment obtenir une clé API Mistral ?

  1. Rendez-vous sur La console Mistral
  2. Créez un compte ou connectez-vous
  3. Allez dans la section « API Keys »
  4. Créez une nouvelle clé et collez-la ci-dessus

Description

Mailbot CEMagik est une extension Thunderbird qui vous assiste dans la rédaction de vos emails grâce aux LLM de Google Gemini ou Mistral AI.

Fonctionnalités

  • Correction orthographique et grammaticale
  • Reformulation avec différents tons
  • Traduction multilingue
  • Analyse et résumé de mails
  • Prompt libre pour interagir avec l'IA
