# Icônes

Placez vos icônes dans ce dossier :

- `icon-48.png` - Icône 48x48 pixels (obligatoire)
- `icon-96.png` - Icône 96x96 pixels (obligatoire)
- `insigneu.png` - Logo Marine (optionnel)
- `ftrh.png` - Logo FTRH (optionnel)

## Créer des icônes rapidement

Si vous n'avez pas d'icônes, vous pouvez :

1. Utiliser un générateur en ligne comme [Favicon.io](https://favicon.io/)
2. Utiliser un emoji comme base
3. Créer avec un outil comme GIMP ou Photoshop

**Format recommandé :** PNG avec fond transparent
